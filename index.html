How to build kernel

$ cd ~
$ git clone https://bitbucket.org/UBERTC/aarch64-linux-android-4.9-kernel.git
$ mkdir sony_kernel && cd $_
$ git clone https://github.com/AndroPlus-org/android_kernel_sony_msm8994.git
$ mkdir final_files #put dtbToolCM, mkbootimg and ramdisk here, then set permission to 755
$ ./build_it4_karin
